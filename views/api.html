ffbinaries API Endpoints


All calls to this API are just regular HTTP GETs, i.e. curl http://ffbinaries.com/api/v1/version/latest.

  • /api/v1 Lists all available ffmpeg versions

  • /api/v1/version/latest Serves the latest available version of ffmpeg (currently 3.4)

  • /api/v1/version/3.2 Permalink to version 3.2

  • /api/v1/version/3.3 Permalink to version 3.3

  • /api/v1/version/3.4 Permalink to version 3.4
