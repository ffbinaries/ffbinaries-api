ffbinaries


Tiny API service which provides links to precompiled binaries for ffmpeg, ffprobe, ffserver and ffplay.

You can connect to this API using regular HTTP GETs, for example curl http://ffbinaries.com/api/v1/version/latest.

There's also a downloader app in Node.js which makes it easy to automatically download the right binaries for your platform or to integrate into your build process.


API Endpoints


  • /api/v1 Lists all available ffmpeg versions

  • /api/v1/version/latest Serves the latest available version of ffmpeg

  • /api/v1/version/3.2 Permalink to a specific version: 3.2
