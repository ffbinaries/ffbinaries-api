ffbinaries


Tiny API service which provides links to precompiled binaries for ffmpeg, ffprobe, ffserver and ffplay.

This service is still in the early phases of development.


  • /api/latest Serves the latest available version of ffmpeg
  • /api/versions Lists all available ffmpeg versions
  • /api/version/3.2 Permalink to ffmpeg 3.2
